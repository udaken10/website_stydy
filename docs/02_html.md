# HTML：文章にタグをつけていこう

HTML（エイチティーエムエル）は、文字の間に「タグ」を入れて、コンピュータに文章の役割を教える言葉です。

## タグの仕組み：サンドイッチをイメージしよう
```html
<p>こんにちは、世界！</p>
```
- `<p>`（開始タグ）：ここから段落だよ、というパン
- `こんにちは、世界！`：中身のレタスやハム
- `</p>`（終了タグ）：ここまで段落だよ、というパン（`/`を忘れずに！）

## よく使うタグ
- `<h1>`: 一番デカい見出し！
- `<p>`: 普通の文章（段落）
- `<a>`: 別のページへの「ワープゲート（リンク）」
- `<ul>` と `<li>`: 箇条書きリスト
