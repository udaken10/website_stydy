# JavaScript：サイトに命を吹き込もう！

JavaScript（ジャバスクリプト）は、サイトに「動き」や「便利な機能」を追加する、魔法のバトンのようなプログラミング言語です。

## どうやって動くの？：3つのステップ
1. **「だれを？」見つける**: 操作したいボタンや文字を探します。
2. **「いつ？」決める**: 「クリックされたら」とか「1秒たったら」とか。
3. **「なにをする？」書く**: 警告を出したり、色を変えたりします。

## main.jsのお仕事
```javascript
const button = document.getElementById('colorButton');

button.addEventListener('click', () => {
    // 背景色をランダムに変えるよ！
});
```
「`colorButton` という名前のボタンを、クリックしたら、背景を変える」という命令になっていますね！

---

### 【クエスト4】あいさつを表示させてみよう
ファイル `scripts/main.js` を開いて、以下のことに挑戦してみよう！

1. `main.js` の一番下に、次のコードを書き足してみて。

```javascript
// ボタンをクリックしたときに「こんにちは！」と表示する
button.addEventListener('click', () => {
    alert('ボタンが押されました！すごい！');
});
```

これで、ボタンを押すと背景色が変わるだけでなく、メッセージが表示されるようになります！
