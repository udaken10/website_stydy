# ウェブサイトの構成って？（中学生向け）

みんなが普段見ているウェブサイトは、実は3つの「役割」分担でできています！

## 1. HTML：サイトの「中身」と「骨組み」
- **たとえ**: 人間の「骨」や、作文の「本文」です。
- **役割**: 「ここがタイトル！」「ここが本文！」といった、情報の意味を決めます。

## 2. CSS：サイトの「オシャレ（装飾）」
- **たとえ**: 人間の「服」や「髪型」、作文の「ペンやシールのデコレーション」です。
- **役割**: 「文字の色を赤に！」「ボタンを真ん中に！」といった、見た目を整えます。

## 3. JavaScript：サイトの「仕掛け（動き）」
- **たとえ**: 人間の「筋肉」や、おもちゃの「電池とモーター」です。
- **役割**: 「ボタンを押したら色が変わる！」「画像をシュッと動かす！」といった、便利な動きを作ります。

---

### 【クエスト1】身近なサイトを観察してみよう
よく使うウェブサイト（YouTubeやSNS、学校のHPなど）を思い浮かべてみてください。

- どこまでが「情報の骨組み（HTML）」かな？
- どんな色や形で「デザイン（CSS）」されているかな？
- どんな「便利な動き（JavaScript）」があるかな？

これらを意識して見てみるのが、プログラミング学習の第一歩です！
