# 準備：VS Codeを「最強の武器」にしよう！

ウェブ制作をはじめる前に、プロが使っている「VS Code」を使いやすく改造しましょう。
これを設定するだけで、プログラミングが10倍楽になります！

## 1. 魔法の道具「Live Server」を入れよう
コードを書き換えたときに、いちいちブラウザの更新ボタンを押すのは大変ですよね。
「Live Server」という道具を入れると、**保存した瞬間にブラウザが自動で書き換わります！**

### 入れかた
1. VS Codeの左側のアイコンから、四角いマーク（拡張機能）をクリック。
2. 上の検索欄に「Live Server」と入力。
3. 紫色のアイコンの「Install」ボタンを押す。

### つかいかた
1. `index.html` を開きます。
2. 画面の右下にある「Go Live」という文字をクリック。
3. 自動でブラウザが立ち上がります！

## 2. 「保存わすれ」をゼロにする設定
「コードを書いたのに反映されない！」という原因の9割は、ファイルの保存忘れです。これを自動化しましょう。

1. VS Codeの左下にある歯車マーク ⚙️ から「Settings」を開く。
2. 一番上のほうにある「Files: Auto Save」を、`off` から **`afterDelay`** に変える。
これで、書いたら勝手に保存されるようになります！

## 3. 画面を横に並べよう
- **左側**: VS Code（書くところ）
- **右側**: ブラウザ（確認するところ）

このように画面を2つ並べて進めるのが、ウェブ制作の「基本スタイル」です！
