# デベロッパーツール：ウェブの裏側をのぞく魔法のメガネ

「このサイト、どうやって作られてるんだろう？」「色が気に入らないから、今すぐ変えてみたい！」
そんな願いをかなえてくれるのが、ブラウザに隠された最強のツール**「デベロッパーツール（開発者ツール）」**です。

これを使いこなせたら、君はもう初心者卒業です！

## 1. 魔法のメガネを起動しよう
ブラウザ（ChromeやEdge）でウェブサイトを開いた状態で、以下のどれかをやってみてね。
- キーボードの **`F12`** を押す。
- 右クリックして **「検証（Inspect）」** を選ぶ。
- `Ctrl + Shift + I` を押す。

すると、右側（または下側）に不思議なコードの画面が出てきます。これがデベロッパーツールの世界です！

## 2. 世界が広がる！デベロッパーツールのすごさ
デベロッパーツールを使うと、**「いま見ているサイトを、こっそり書き換える」**ことができます。
（※自分の画面で見えている分が変わるだけで、元のサイトは壊れないから安心してください！）

- **色を変え放題**: 好きなアイドルのサイトの背景をピンクにしたり、文字を巨大にしたりできます。
- **配置をチェック**: どうやって画像が並んでいるか、余白（Margin）がどれくらいか丸見えです。
- **エラーを教えてくれる**: JavaScriptが動かないとき、どこが間違っているか教えてくれます。

## 3. 基本的な使い方

### ① 要素（Elements）タブ：骨組みを見る・変える
1. デベロッパーツールの左上にある **矢印マーク（左上のアイコン）** をクリック。
2. そのままサイトの好きな場所（文字や画像）をクリックしてみて！
3. プログラムの場所がハイライトされます。
4. そのコードをダブルクリックして書き換えると……**目の前のサイトの文字が今すぐ変わります！**

### ② スタイル（Styles）タブ：オシャレを微調整
「要素」タブのすぐ下に「Styles」という場所があります。
- ここに書かれている色（color）や大きさ（font-size）の数字をいじってみよう。
- リアルタイムでデザインが変わるから、CSSの練習に最高です。

### ③ コンソール（Console）タブ：JavaScriptとおしゃべり
ここにはプログラムからのメッセージが表示されます。
- エラーが出ているときは、ここで赤色の文字が表示されます。
- `console.log("こんにちは");` と書くと、ここに文字が出せます。

## 4. クエスト：デベロッパーツールで遊ぼう！
この教材の `index.html` を開いて、デベロッパーツールで以下に挑戦してね。

1. **タイトルをハックせよ！**: `<h1>`の中身を「自分の名前」に書き換えてみて。
2. **色を盗め！**: ヘッダーの色を、自分が一番好きな色にその場で書き換えてみて。
3. **隠しメッセージ？**: コンソールタブを開いて、何かエラーが出ていないか、またはメッセージが出ていないか確認してみて。
