# CSS：サイトに魔法の粉をかけよう

CSS（シーエスエス）は、HTMLで作った骨組みを「カッコよく」「かわいく」変身させる言葉です。

## 書き方のルール：3ステップ
```css
h1 {
    color: red;
}
```
1. **だれが？** (`h1`): 見出しを
2. **なにを？** (`color`): 文字の色を
3. **どうする？** (`red`): 赤にする！

## ボックスモデル：みんな四角い
ウェブサイトの要素は、全部「透明な四角い箱」に入っていると考えます。
- **Padding（中身のまわり）**: 箱の中のゆとり
- **Border（フチ）**: 箱の枠線
- **Margin（外がわ）**: 隣の要素との距離

### 【クエスト3】見た目を変えてみよう
ファイル `styles/style.css` を開いて、以下のことに挑戦してみよう！

1. `header` という部分を探して、`background-color` を好きな色（例えば `green` や `#ff6600` など）に変えてみて！
2. `h1` の文字をもう少し大きくするために、`header` の中に `font-size: 2.5rem;` を書き足してみよう。

```css
header {
    background-color: #ff6600; /* ここを変える */
    color: white;
    padding: 1rem;
    font-size: 2.5rem; /* これを書き足す */
    /* ... */
}
```

色が変わったら、君のオリジナルサイトへの第一歩だ！
